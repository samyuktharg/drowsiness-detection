# Drowsiness Detection System

A real-time safety system that detects driver drowsiness using computer vision.

## How to Run
1. Create a virtual environment:
   ```bash
   python -m venv venv
   .\venv\Scripts\activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the code:
   ```bash
   python inference.py
   ```
